# FAST AS SH-T Store

## Descripción

Marketplace deportivo estático con landing page, catálogo de productos, sección de eventos y contacto. Proyecto enfocado en aplicar principios de diseño responsive, componentes reutilizables y coherencia visual de marca.

## Tecnologías

- **HTML5**
- **CSS3** (Grid, Flexbox, Responsive Design)

## Características

- Landing page con hero section
- Catálogo de productos
- Sección de eventos
- Formulario de contacto
- Componentes reutilizables (cards, banners, footer)
- Diseño responsive (desktop, tablet, mobile)
- Paleta y estilo de marca F.A.S.T
